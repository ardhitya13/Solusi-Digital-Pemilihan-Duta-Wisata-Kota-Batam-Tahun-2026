# PSTEAM Fullstack Project
## Kelompok 7 - Proyek PSTEAM

Website resmi PSTEAM (Polibatam Software Team) yang dikembangkan sebagai proyek fullstack menggunakan Next.js pada sisi frontend serta Node.js + Express + Prisma pada sisi backend.  
Proyek ini dibuat untuk memenuhi tugas mata kuliah PBL.

## Proyek Duta Pariwisata Batam

- Kode Tim: IF-4MA-02
- Judul: Solusi Digital Pemilihan Duta Wisata Kota Batam Tahun 2026

### Deskripsi Permasalahan
Solusi Digital Pemilihan Duta Wisata Kota Batam Tahun 2026 berupa aplikasi Pemilihan Encik Puan Duta Wisata Kota Batam 2026 berbasis web dirancang untuk membantu Pemerintah Kota Batam, khususnya Dinas Kebudayaan dan Pariwisata, dalam mengelola proses secara terintegrasi, efisien, dan terdokumentasi. Melalui sistem ini, panitia dapat melakukan pendataan pendaftaran peserta secara online (biodata, unggah berkas persyaratan, dan status verifikasi), melaksanakan penilaian oleh dewan juri menggunakan kriteria yang terukur dan transparan, serta menghasilkan rekapitulasi nilai otomatis hingga laporan akhir peringkat peserta. Dengan alur kerja yang terstandar, solusi ini diharapkan meningkatkan akurasi data, mengurangi duplikasi dan kesalahan input, mempercepat proses evaluasi, serta menyediakan arsip dan laporan yang rapi untuk kebutuhan monitoring, pelaporan, dan pengambilan keputusan pada penyelenggaraan Pemilihan Duta Wisata Kota Batam Tahun 2026.
